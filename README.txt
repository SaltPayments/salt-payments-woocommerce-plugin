Install and activate WooCommerce
Upload the zip file through Plugins -> Add New -> Upload plugin
Activate the plugin through the Plugins menu in WordPress
Go to WooCommerce -> Settings -> Checkout -> Salt Payments
Select 'Enable SALT Payments Gateway Module'
Edit title, which controls the heading that the user sees during checkout, for example set "Pay with Credit Card"
Set your Merchant ID and API Token
Select either Sandbox or Production mode.
Save changes.
